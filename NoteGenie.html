NoteGenie - Case Study

A mobile app that turns educational videos into instant notes and summaries using AI.

Project Overview

NoteGenie is a productivity app designed to help students and learners generate notes and summaries from any video content in seconds. With an intuitive and clean UI, the app makes it easy to process video content and extract meaningful study materials on the go.

Problem Statement

Students often struggle to take notes while watching educational videos. Switching between apps, missing key points, or forgetting to summarize after watching makes the learning process inefficient.

Objective

Build an AI-powered mobile UI that allows users to search for videos, convert them into notes or summaries instantly, and organize the content efficiently for future reference.

Target Users

  • Students and online learners.
  • Professionals attending webinars and lectures.
  • Content reviewers and educators.

Design Process

  • Research: Reviewed productivity and note-taking apps, identified user pain points.
  • Wireframes: Simple, card-based UI for summaries, video input, and notes.
  • UI Design: Purple gradient theme with rounded buttons and soft UI elements.
  • Prototype: Full walkthrough of login, home, search, summary generation, saved data, and user profile screens.

Challenges

Creating a smooth, minimal UX that balances both simplicity and functionality while representing AI features clearly.

Key Learnings

I learned how effective visual hierarchy, minimal navigation, and proper data grouping can drastically enhance usability—especially in AI-powered tools where instant results are expected.
